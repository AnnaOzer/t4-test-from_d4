в папке t4/framework/4Mvc/Extensions
расширение это библиотека внутри фреймворка
это библиотеки для вью вроде бутстрапа и капчи и джиквери
это попытка интегрировать хорошо известные библиотеки для вью внутрь фреймворка

внутри папки расширения есть файл Extension.php
это класс расширения

этот класс должени реализовать всего один метод init()
обычно в этом методе публикуются ресурсы
например бутстрап, метод инит вначале получает конфиг расширения
